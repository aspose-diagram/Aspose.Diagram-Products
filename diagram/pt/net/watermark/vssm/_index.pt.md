Arquivos com extensão .VSSM são Microsoft Visio arquivos de estêncil que oferecem suporte para macros. Em geral, Microsoft Visio é um software de desenho que permite criar arquivos que podem conter e representar informações definidas pelo usuário em diferentes formas.

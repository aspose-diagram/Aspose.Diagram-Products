XAML, Genişletilebilir Uygulama İşaretleme Dili, uzantı dosyaları, Windows Presentation Foundation (WPF) tabanlı yazılım uygulamaları için kullanıcı arabirimi öğelerini tanımlar. Bir dil olmasına rağmen, kullanımı ve anlaşılması kolay standart XML formatına dayandığından programlanması gerekmez. XAML ("zammel" olarak telaffuz edilir), belirli kullanıcı arayüzleri oluşturmak amacıyla Microsoft tarafından geliştirilmiştir. Orijinal kısaltması, Avalon'un WPF'nin kod adı olduğu Genişletilebilir Avalon İşaretleme Dili anlamına geliyordu. XAML dosyaları bazen XOML uzantısıyla da kaydedilir.

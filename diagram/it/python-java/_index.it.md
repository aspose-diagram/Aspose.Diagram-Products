<p>
 Aspose.Diagram per Python tramite Java è una soluzione di manipolazione dei documenti Microsoft Visio. È possibile leggere, scrivere, proteggere, esportare e stampare formati Visio nativi, nonché supportare alcuni formati di immagine e layout fisso comunemente usati. È facile caricare i file esistenti, modificare gli elementi diagram per esportarli nel formato desiderato, comprese le immagini o formati a layout fisso come PDF o XPS.
</p>

<p>
 Aspose.Diagram per Python tramite Java è un API autonomo e non richiede Microsoft Visio o qualsiasi altra installazione di software. Inoltre, poche attività che si possono eseguire facilmente come creare diagrammi da zero, leggere o scrivere vari formati, accedere e leggere le proprietà incluse Visio formule oggetto, esportare diagrammi in più formati, stampare diagrammi ad alta fedeltà e molto altro.
</p>

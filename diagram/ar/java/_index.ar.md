<p>
 Aspose.Diagram for Java هو Microsoft Visio معالجة مستندات API فعالة. يوفر وظائف مشتركة مثل الإنشاء والتحليل & amp؛ تحويل التنسيقات الأصلية Visio بالإضافة إلى دعم بعض تنسيقات الصور والتخطيط الثابت شائعة الاستخدام. يمكن للتطبيقات تحميل المستندات الموجودة ومعالجة عناصر diagram لتصدير النتيجة في التنسيقات الخاصة بها أو الصور أو التنسيقات ذات التخطيط الثابت.
